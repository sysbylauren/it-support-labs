## 🛠️ Lab Write-Up Structure
Each lab directory contains a dedicated `README.md` formatted with:
* **Scenario Overview:** Problem statement and user request.
* **Troubleshooting Steps:** Logical flow taken to diagnose the issue.
* **Resolution:** Final solution and verified outcome.
* **Key Takeaways & Screenshots:** Visual proof and practical lessons learned.
